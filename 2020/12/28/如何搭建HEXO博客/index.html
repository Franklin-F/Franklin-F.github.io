如何搭建HEXO博客

Created2020-12-28|Updated2020-12-28
|Post View:

1.安装软件

1.1Git

1.2Node.js

不能下载,下载很慢的童鞋请在百度盘下载

链接：https://pan.baidu.com/s/18l8HE8spcNAvnZ23n9_Fsw
提取码：ngwg

2.安装HEXO

使用git bash输入

安装完成后找一个自己的文件夹我这里使用的是D:\根目录

然后用hexo初始化一下

本地先看一哈

之后访问Http://localhost:4000就可以看到本地站点了使用ctrl+C停止

3.新建Github仓库

创建一个和你用户名相同的仓库，后面加.github.io，只有这样，将来要部署到 GitHub page 的时候，才会被识别， 也就是 xxxx.github.io，其中 xxx 就是你注册 GitHub 的用户名。我这里是已经建过了。

5.配置用户和邮箱

6. 将 hexo 部署到 GitHub

这一步，我们就可以将 hexo 和 GitHub 关联起来，也就是将 hexo 生成的文章部署到 GitHub 上，打开站点配置文件 _config.yml，翻到最后，修改为
YourgithubName 就是你的 GitHub 账户

这个时候需要先安装 deploy-git ，也就是部署的命令，这样你才能用命令部署到 GitHub。

然后

注意 deploy 时可能要你输入 username 和 password。github的用户名和密码。
